## 실습으로 배우는 선착순 이벤트 시스템
노션 링크 : 
https://dynamic-cent-685.notion.site/116ed6ed7f7c807582e5c514cb5ca7f0

### 전체 요약

- 쿠폰 선착순 100장 발급 로직 구현 시 발생되는 동시성 문제 해결
- 여러 스레드가 동시에 쿠폰 발급 요청 시 쿠폰 100장 초과 발급 문제
    - → Redis Incr을 활용하여 해결. incr로 count 조회 후 100미만일 경우만 실행.
- 발급 요청 과부하로 인한 서비스 장애 문제
    - → 동시에 몰린 쿠폰 발급요청을 kafka로 발송하여 순차적으로 처리.
- 한 사람에게 쿠폰이 중복 발급 되는 문제
    - → redis set 자료구조 활용. 발급 userid 기록 (RDB를 사용하는 방법도 고려)
